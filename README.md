## Infinite recursive shape

### Installation

```bash
git clone https://github.com/basemkhirat/recursive-matrix
cd recursive-matrix
npm install
npm start
```
Development server will be started at port 3000.

Open [http://localhost:3000](http://localhost:3000) to view the react app.

### Testing

```bash
npm test
```

### Time Complexity: O(1)
